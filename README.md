# ⚡ EnergyBot - Assistente Virtual de Concessionária de Energia

Este é um projeto didático em Python desenvolvido para simular o fluxo de atendimento automatizado de uma concessionária de energia elétrica comercial. Ideal para portfólio de estudo em lógica de programação e automações estruturadas de menus.

## 🚀 Funcionalidades Demonstradas
* **Identificação segura:** Busca de dados de clientes simulada a partir do CPF (com limpeza de caracteres).
* **Faturamento Inteligente:** Geração de segunda via e linha digitável para pagamento de faturas pendentes.
* **Abertura de Chamados:** Registro de falta de energia com geração dinâmica de protocolo de atendimento técnico.
* **Consulta de Dados:** Exibição do histórico de consumo de energia de forma dinâmica.

## 🛠️ Tecnologias Utilizadas
* **Python 3.10+** (Lógica estruturada, manipulação de coleções de dados, simulação de tempo de resposta).
